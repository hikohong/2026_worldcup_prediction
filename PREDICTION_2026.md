## 重點發現

- **阿根廷大幅躍升至 32.9%**：完美近況（5 勝，進 16 失 1）× 最輕鬆分組（Group J）× Scaloni 最高教練評分（9.5）× 美洲主場，四項因素同時最優
- **法國維持第二（24.7%）**：FIFA 排名第一但 Camavinga 落選略有影響；Deschamps 最後一屆帶動凝聚力
- **德國超越巴西（15.0% vs 14.0%）**：德國 5/5 全勝、Wirtz+Musiala 雙核爆發；巴西三大主力傷缺重創陣容
- **巴西下滑至第四（14.0%）**：Militão、Rodrygo、Estêvão 同時傷缺是最大警號；Ancelotti 磨合期仍短
- **西班牙第五（13.4%）**：Yamal + Nico Williams 雙傷為主要風險；De la Fuente 戰術底蘊仍強
- **義大利❌出局**：連續三屆（2018/2022/2026）未出線，主要官員相繼辭職

## 預測方法架構

```
最終分數 = 歷史分數 × 綜合調整倍率

歷史分數  = ( Exp(α=0.3) + MA(4) + MA(3) ) / 3
              ↑ 近10屆60筆回測 MSE 最低的前三方法

綜合倍率  = (陣容 / μ陣容) × 抽籤× × (教練 / μ教練) × 主場×
              ↑ 各因素以5隊重新正規化，全隊平均倍率 ≈ 1.0
```
