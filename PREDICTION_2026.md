## 預測方法架構

```
最終分數 = 歷史分數 × 綜合調整倍率

歷史分數  = ( Exp(α=0.3) + MA(4) + MA(3) ) / 3

              ↑ 各因素正規化後相乘，全隊平均倍率 ≈ 1.0
```
