# 2026 FIFA World Cup Prediction

用歷史資料與統計演算法預測 2026 FIFA 世界盃奪冠機率。
涵蓋線性回歸、移動平均、指數平滑化等多種方法，並以近10屆回測驗證準確度。

> 📄 **最新預測報告（含即時球隊現況）→ [PREDICTION_2026.md](PREDICTION_2026.md)**

## 演算法

### 基礎三種（code_19_1 ~ 3）

| 演算法 | 公式 | 時間複雜度 |
|--------|------|-----------|
| **線性回歸** | $y = ax + b$，最小化 $\sum(y_i - \hat{y}_i)^2$ | O(N) |
| **移動平均 MA(k)** | $\hat{y}_{t+1} = \frac{1}{k}\sum_{j=0}^{k-1} y_{t-j}$ | O(N) |
| **指數平滑化 Exp(α)** | $S_t = \alpha y_t + (1-\alpha)S_{t-1}$ | O(N) |

### 擴充三種（test_new_methods.cpp）

| 演算法 | 特色 |
|--------|------|
| **加權線性回歸** | 指數衰減權重 $w_i = r^{n-1-i}$（r=0.85），越近期資料權重越大 |
| **多項式回歸（二次）** | $y = c_2x^2 + c_1x + c_0$，高斯消去法求解 3×3 正規方程組 |
| **自動調參指數平滑** | 三分搜尋找最佳 α，最小化交叉驗證 MSE |

## 回測結果（近10屆，1986–2022）

以 **6隊 × 10屆 = 60筆預測** 評估各方法準確度：

| 排名 | 方法 | MSE | MAE |
|------|------|-----|-----|
| 🥇 1 | **Exp(α=0.3)** | **5.82** | 2.05 |
| 🥈 2 | MA(4) | 6.18 | 2.15 |
| 🥉 3 | MA(3) | 6.19 | 2.11 |
| 4 | Exp(α=0.5) | 6.35 | 2.09 |
| 5 | 自動調參 ES | 6.40 | 2.12 |
| 6 | 線性回歸(w=10) | 6.82 | 2.16 |
| 7 | 加權線性回歸 | 7.00 | 2.14 |
| 8 | 多項式回歸 | 10.20 | 2.49 |

> 最簡單的 **Exp(α=0.3)** 回測 MSE 最低。多項式回歸因10筆資料容易過擬合，MSE 最高。

## 如何執行

需要支援 C++17 的編譯器（g++ 7 以上）。

```bash
# 基礎演算法範例
g++ -std=c++17 -O2 -o code_19_1 codes/chap19/code_19_1.cpp && ./code_19_1

# 2026 自動模型選擇預測
g++ -std=c++17 -O2 -o code_19_4 codes/chap19/code_19_4.cpp && ./code_19_4

# 三種新演算法（加權線性回歸 / 多項式 / 自動調參 ES）
g++ -std=c++17 -O2 -o test_new codes/chap19/test_new_methods.cpp && ./test_new

# 8方法近10屆回測報告
g++ -std=c++17 -O2 -o backtest codes/chap19/backtest_10.cpp && ./backtest

# 前三方法 2026 預測
g++ -std=c++17 -O2 -o predict codes/chap19/predict_2026_top3.cpp && ./predict

# 完整因素預測（含球員陣容、抽籤、教練、主場）
g++ -std=c++17 -O2 -o predict_full codes/chap19/predict_2026_full.cpp && ./predict_full

# 單元測試（37 個，全通過即 exit 0）
g++ -std=c++17 -O2 -o test_chap19 codes/chap19/test_chap19.cpp && ./test_chap19

# Loop 測試（7群組，37 個）
g++ -std=c++17 -O2 -o test_loop codes/chap19/test_loop.cpp && ./test_loop
```

## 注意事項

`predict_2026_full.cpp` 與 `PREDICTION_2026.md` 已納入以下因素：
- ✅ 球員狀態與陣容深度（含傷兵更新）
- ✅ 賽程抽籤與分組（依實際分組）
- ✅ 教練戰術評分
- ✅ 主場優勢（2026 由美國、加拿大、墨西哥聯合主辦）

純歷史統計模型（`code_19_4.cpp`、`predict_2026_top3.cpp`）不含上述因素，僅供回測比較使用。
