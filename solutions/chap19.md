# 19 章　未来予測アルゴリズム

## 概要

過去の観測データから未来の値を予測する手法を扱います。本章では代表的な3つのアプローチを取り上げます。

---

## 19.1　線形回帰 (最小二乗法)

**アイデア:** N 個のデータ点 (x_i, y_i) に対して，誤差の二乗和

$$E(a, b) = \sum_{i=1}^{N} (y_i - (a x_i + b))^2$$

を最小化する直線 y = ax + b を求める。偏微分を 0 とおいて解くと

$$a = \frac{N \sum x_i y_i - \sum x_i \sum y_i}{N \sum x_i^2 - (\sum x_i)^2}, \quad b = \frac{\sum y_i - a \sum x_i}{N}$$

が得られる。計算量は O(N)。

**入力例**
```
5 6
1 2.1
2 4.0
3 5.9
4 8.1
5 9.8
```

**出力例**
```
回帰直線: y = 1.93 * x + 0.14
x = 6 のときの予測値: 11.72
```

---

## 19.2　移動平均

**アイデア:** 直近 k 個の観測値の平均を次の時刻の予測値とする。

$$\hat{y}_{t+1} = \frac{1}{k} \sum_{j=0}^{k-1} y_{t-j}$$

窓幅 k が大きいほど滑らかな予測になるが，急激な変化への反応が遅れる。計算量は O(N k)（スライディングウィンドウを用いれば O(N)）。

**入力例**
```
6 3
10 20 30 40 50 60
```

**出力例**
```
移動平均 (窓幅 k = 3):
  区間 [0, 2] の平均: 20
  区間 [1, 3] の平均: 30
  区間 [2, 4] の平均: 40
  区間 [3, 5] の平均: 50
次のステップの予測値: 50
```

---

## 19.3　指数平滑化

**アイデア:** 新しい観測値ほど大きな重みを与えながら平均を取る。

$$S_t = \alpha \, y_t + (1 - \alpha) S_{t-1}, \quad S_0 = y_0$$

平滑化係数 α ∈ (0, 1) が大きいほど最新のデータを重視し，小さいほど過去のデータを重視する。計算量は O(N)。

**入力例**
```
5 0.3
10 20 15 25 30
```

**出力例**
```
指数平滑化 (alpha = 0.3):
  S[0] = 10
  S[1] = 13
  S[2] = 13.5
  S[3] = 16.95
  S[4] = 20.865
次のステップの予測値: 20.865
```

---

## 各手法の比較

| 手法 | 計算量 | 特徴 |
|------|--------|------|
| 線形回帰 | O(N) | データにトレンド (傾き) がある場合に有効 |
| 移動平均 | O(N) | シンプルで実装が容易；ノイズ除去に強い |
| 指数平滑化 | O(N) | 直近データを重視；α で感度を調整できる |

## 19.4　応用例：FIFA ワールドカップ 2026 優勝予測

### データ設計

1930〜2022 年の全 22 大会について，主要 6 チームの「表現スコア」を定義する。

| 結果 | スコア |
|------|--------|
| 優勝 | 7 |
| 準優勝 | 6 |
| 3 位 | 5 |
| 4 位 | 4 |
| ベスト8 | 3 |
| ベスト16 | 2 |
| グループ敗退 | 1 |
| 不参加・予選落ち | 0 |

### モデル選択 (クロスバリデーション)

後 5 大会 (2002〜2022) を検証データとして，各モデルの MSE を比較した結果：

| チーム | 最良モデル | MSE |
|--------|-----------|-----|
| Brazil | 線形回帰 | 1.97 |
| Germany | Exp(α=0.3) | 7.02 |
| Argentina | 線形回帰 | 4.17 |
| France | 線形回帰 | 7.30 |
| Italy | MA(3) | 7.00 |
| Spain | MA(4) | 5.85 |

### 2026 予測結果

チームごとに最良モデルで予測したスコアを正規化して優勝確率に変換：

| 順位 | チーム | 優勝確率 |
|------|--------|---------|
| 1 | **Argentina** | **23.3%** |
| 2 | **France** | **22.9%** |
| 3 | Brazil | 19.2% |
| 4 | Spain | 16.5% |
| 5 | Germany | 16.4% |
| 6 | Italy | 1.7% |

### 考察

- **線形回帰**がブラジル・アルゼンチン・フランスに最適：近年の一貫したトレンドを捉えている
- **指数平滑化**がドイツに最適：2018/2022 のグループ敗退という急激な下落に反応
- **移動平均**がイタリア・スペインに最適：予選落ちを含む不規則な変動に対して安定
- イタリアは 2018・2022 連続予選落ちで MA(3) 予測値が急落し，確率 1.7% に留まる

> **注意**: 本予測は過去の表現スコアのみに基づく統計モデルです。選手コンディション，組み合わせ抽選，戦術的要因は一切考慮していません。
