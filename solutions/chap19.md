# 第19章　未來預測演算法

## 概述

本章介紹從過去的觀測資料預測未來值的方法，涵蓋三種具代表性的方法。

---

## 19.1　線性回歸（最小二乘法）

**核心概念：** 對 N 個資料點 (x_i, y_i)，最小化誤差平方和

$$E(a, b) = \sum_{i=1}^{N} (y_i - (a x_i + b))^2$$

求直線 y = ax + b。對偏微分令其為 0 後求解可得：

$$a = \frac{N \sum x_i y_i - \sum x_i \sum y_i}{N \sum x_i^2 - (\sum x_i)^2}, \quad b = \frac{\sum y_i - a \sum x_i}{N}$$

時間複雜度為 O(N)。

**輸入範例**
```
5 6
1 2.1
2 4.0
3 5.9
4 8.1
5 9.8
```

**輸出範例**
```
回歸直線: y = 1.93 * x + 0.14
x = 6 時的預測值: 11.72
```

---

## 19.2　移動平均

**核心概念：** 以最近 k 個觀測值的平均作為下一時刻的預測值。

$$\hat{y}_{t+1} = \frac{1}{k} \sum_{j=0}^{k-1} y_{t-j}$$

視窗大小 k 越大，預測越平滑，但對急劇變化的反應越遲鈍。時間複雜度為 O(Nk)（使用滑動視窗可達 O(N)）。

**輸入範例**
```
6 3
10 20 30 40 50 60
```

**輸出範例**
```
移動平均 (視窗大小 k = 3):
  區間 [0, 2] 的平均: 20
  區間 [1, 3] 的平均: 30
  區間 [2, 4] 的平均: 40
  區間 [3, 5] 的平均: 50
下一步的預測值: 50
```

---

## 19.3　指數平滑化

**核心概念：** 對較新的觀測值給予較大的權重進行加權平均。

$$S_t = \alpha \, y_t + (1 - \alpha) S_{t-1}, \quad S_0 = y_0$$

平滑係數 α ∈ (0, 1) 越大，越重視最新資料；越小，越重視歷史資料。時間複雜度為 O(N)。

**輸入範例**
```
5 0.3
10 20 15 25 30
```

**輸出範例**
```
指數平滑化 (alpha = 0.3):
  S[0] = 10
  S[1] = 13
  S[2] = 13.5
  S[3] = 16.95
  S[4] = 20.865
下一步的預測值: 20.865
```

---

## 各方法比較

| 方法 | 時間複雜度 | 特點 |
|------|-----------|------|
| 線性回歸 | O(N) | 資料具有趨勢（斜率）時效果佳 |
| 移動平均 | O(N) | 簡單易實作；對雜訊有較強的抵抗力 |
| 指數平滑化 | O(N) | 重視近期資料；可透過 α 調整靈敏度 |

## 19.4　應用範例：FIFA 世界盃 2026 奪冠預測

### 資料設計

針對 1930〜2022 年的全部 22 屆賽事，定義主要 6 支隊伍的「表現分數」。

| 結果 | 分數 |
|------|------|
| 冠軍 | 7 |
| 亞軍 | 6 |
| 第三名 | 5 |
| 第四名 | 4 |
| 八強 | 3 |
| 十六強 | 2 |
| 小組出局 | 1 |
| 未參賽・預選賽落敗 | 0 |

### 模型選擇（交叉驗證）

以最後 5 屆（2002〜2022）作為驗證資料，比較各模型 MSE 的結果：

| 隊伍 | 最佳模型 | MSE |
|------|---------|-----|
| Brazil | 線性回歸 | 1.97 |
| Germany | Exp(α=0.3) | 7.02 |
| Argentina | 線性回歸 | 4.17 |
| France | 線性回歸 | 7.30 |
| Italy | MA(3) | 7.00 |
| Spain | MA(4) | 5.85 |

### 2026 預測結果

以各隊的最佳模型預測分數，正規化後轉換為奪冠機率：

| 排名 | 隊伍 | 奪冠機率 |
|------|------|---------|
| 1 | **Argentina** | **23.3%** |
| 2 | **France** | **22.9%** |
| 3 | Brazil | 19.2% |
| 4 | Spain | 16.5% |
| 5 | Germany | 16.4% |
| 6 | Italy | 1.7% |

### 分析

- **線性回歸**最適合巴西、阿根廷、法國：能捕捉近年一致的上升趨勢
- **指數平滑化**最適合德國：對 2018/2022 小組出局的急劇下滑有敏銳反應
- **移動平均**最適合義大利、西班牙：對含預選賽落敗的不規則波動較為穩定
- 義大利因 2018、2022 連續預選賽落敗，MA(3) 預測值急跌，奪冠機率僅 1.7%

> **注意**：本預測僅為基於過去表現分數的統計模型。未考慮球員狀態、賽程抽籤及戰術等因素。
